Leaflet

    Leaflet es una biblioteca en JavaScript para el desarrollo de aplicaciones geoespaciales en la Web.

    Está diseñada para ser liviana y por lo tanto apropiada para ser usada en dispositivos móviles.

    Junto con Google Maps y OpenLayers, es una de las bibliotecas más populares para la programación de mapas para la Web.

Principales características de Leaflet

    Permite desplegar mapas de mosaico (tiled web maps) (ej. OpenStreetMap, Google Maps) como base y agregar capas adicionales con datos del usuario en diferentes formatos (ej. GeoJSON, KML).

    Sus capacidades básicas son extensibles a través de complementos (plugins) programados por la comunidad de desarrolladores.

La clase Map

    La clase Map es la clase central de Leaflet. Se utiliza para crear un mapa y manipularlo.

    El constructor recibe como parámetros:

    • El identificador (id) del elemento HTML (ej. div) que contiene el mapa.
    • Una lista de opciones.

Ejemplo de uso de la clase Map


              // Inicializa un mapa en el elemento "map" 
              // con un centro y un nivel de acercamiento iniciales.
              var map = L.map('map', {
                center: [10, -84],      
                zoom: 15
              });

La clase TileLayer

    La clase TileLayer se utiliza para cargar y desplegar capas basadas en teselas (ej. OpenStreetMap, Mapbox).

    El constructor recibe como parámetros:

    • Una plantilla de URL.
    • Una lista de opciones.

Ejemplo de uso de la clase TileLayer


              // Crea una capa de tipo "TileLayer" basada en OpenStreetMap,
              // especifica la opción "attribution"
              // y la añade al mapa "map" con el método "addTo()".
              L.tileLayer('http://{s}.tile.osm.org/{z}/{x}/{y}.png', {
                attribution: 'OpenStreetMap'
              }).addTo(map);

Proveedores de capas tipo TileLayer

    El complemento Leaflet-providers permite agilizar el uso de capas tipo TileLayer.

La clase GeoJSON

    La clase GeoJSON representa un objeto o un arreglo de objetos de tipo GeoJSON, los cuales pueden ser desplegados como una capa en un mapa.

    El constructor recibe como parámetros:

    • Un objeto en el formato GeoJSON.
    • Una lista de opciones.

Ejemplo de uso de la clase GeoJSON


              // Crea una capa de tipo GeoJSON y especifica un estilo.
              L.geoJson(datosGeoJSON, {
                style: function (feature) {
                  return {
                    fillColor: '#0000ff', 
                    color: '#ffffff'
                  };
                }
              }).addTo(map);
